# 简历星 Pro

一个纯前端的中文简历生成器，单文件，无需安装，浏览器直接打开即用。

## 快速使用

直接双击打开 `index.html`，或用浏览器访问即可。

## 功能特性

- 📄 **简历管理**：多份简历管理，卡片缩略图预览，自动保存到浏览器本地
- 🎨 **20 套模板**：专业商务 / 极简简约 / 创意设计 / 学术科研 / 科技互联网
- ✏️ **完整编辑**：10+ 栏目，工作/教育/技能/项目/证书/语言等
- ⚡ **拖拽排序**：栏目顺序、条目顺序均支持拖拽调整
- 🗑️ **删除管理**：所有栏目和条目均可删除
- ★ **星号技能**：五星熟练度，无进度条
- 👁️ **模板预览**：悬停左侧列表即时预览，支持"试用"和"确认选择"
- 🤖 **AI 诊断**：调用 Claude API 进行五维评分 + 修改建议
- 🖨️ **PDF 导出**：浏览器打印 → 另存为 PDF

## 技术栈

- React 18（UMD CDN）
- Babel Standalone（浏览器编译 JSX）
- 无构建步骤，无 Node.js 依赖
- 数据存储：localStorage（本地浏览器）

## AI 诊断说明

AI 诊断功能调用 Anthropic Claude API（`claude-sonnet-4-20250514`）。
在 claude.ai 中打开时自动生效；在本地浏览器打开时需要注意跨域限制。
